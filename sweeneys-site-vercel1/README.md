This is a Next.js + Tailwind site with 3 sections: About, Menu, Events.
Events are embedded from the official Facebook Page Plugin (Events tab).

## Local preview
1. Install Node.js 18+
2. In this folder:
   - npm install
   - npm run dev
3. Open http://localhost:3000

## Deploy to Vercel
- Create a new project on Vercel and import this folder (upload the zip or connect a repo).
- Framework: Next.js (auto-detected)
- Build command: `next build` (default)
- Output: Next.js (default)
